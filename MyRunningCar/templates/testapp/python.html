COURSE INFO

Python is a general purpose high level programming language,which was developed by Guido Van Rossam in 1989 while working at National Research Institute at Netherlands. But officially Python was made available to public in 1991. Python is a platform independent language can run on any platform without rewriting once again. It’s not surprising for it is an interpreted language; users can directly run the program without compiling data into machine language before execution. Python is living up to its name when it comes to cybersecurity fierceness. With powerful third-party libraries (e.g., Nmap, Yara, Requests, etc.) to choose from. We can use python almost all the purpose as Web Development, Machine Learning, Data Mining, Text Mining, Deep learning ..etc which is a cleare indication towards its future.
