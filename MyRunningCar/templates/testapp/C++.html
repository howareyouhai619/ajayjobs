C++ is a general purpose programming language developed by Bjarne Stroustrup at Bell labs in 1979, as an extension to the C language and widely used now a days for competitive programming. C++ is an object-oriented programming language. It is an extension to C programming. C++ allows you to have a lot of control as to how you use computer resources, so in the right hands its speed and ability to cheaply use resources should be able to surpass other languages. StackOverflow is a programming Q&A site you will no doubt become intimate with as a coding beginner. C++ is a cross-platform language that can be used to create high-performance applications. It has imperative, object-oriented and generic programming features.
